# 周报：批量银行报盘合并与工资表打印工具 (2026/07/06-07/10)

## 一、项目概况

开发了一个带 GUI 的 Python 批量打印工具，核心功能：

1. **银行报盘合并** — 自动识别工商/建设/吉林 3 种银行格式，统一映射为建行 9 列格式，按单位合并
2. **工资表打印** — 按合并文件排序顺序，通过 WPS COM 自动化批量打印已签字的工资表
3. **多工资表按个税合并** — 将多单位工资表合并为"有个税"/"无个税"两张表，拆分对应银行报盘
