# 源文件表头合并规则分析

参考文件：`signed_吉林大学白求恩医学部医学教育办公室2026年06月工资表.xlsx`

---

## 总体布局

```
         Col A-P (1-16)  │ Col Q-AA (17-27)      │ Col AB-AD (28-30)
Row 1:   [标题 C1:AD1]
Row 2:   [单位名称 B2:M2] │ ...                   │ [填报时间 T2:AD2]
Row 3:   序号~公积金基数    │ 扣款明细 (Q3:AA3)      │ 个人所得税~实发合计
Row 4:   (垂直合并至 R3)  │ 养老│失业│医疗│工伤│公积金│... │ (垂直合并至 R3)
Row 5:   (垂直合并至 R3)  │ 单位│个人│单位│个人│...    │ (垂直合并至 R3)
```

---

## 合并规则详解

### 规则 1：单层列 — 垂直合并 {col}3:{col}5

**适用条件**：列名不含 `/`，没有子结构。

**效果**：Row 3 显示值，Row 4 = `None`，Row 5 = `None`（被合并吸收）。

**涉及列**：A-P, AB-AD

```
A3:A5 → 序号     B3:B5 → 姓名     C3:C5 → 身份证
D3:D5 → 部门     E3:E5 → 岗位     F3:F5 → 职工号
G3:G5 → 基本工资  H3:H5 → 交通补贴  I3:I5 → 应发工资
J3:J5 → 单位缴纳五险一金   K3:K5 → 单位代理费
L3:L5 → 转账合计  M3:M5 → 社保基数  N3:N5 → 医保基数
O3:O5 → 工伤基数  P3:P5 → 公积金基数
AB3:AB5 → 个人所得税  AC3:AC5 → 实发工资  AD3:AD5 → 实发合计
```

**代码**：
```python
# 对 canonical_cols 中不含 "/" 的列，垂直合并
if "/" not in cname:
    col_letter = get_column_letter(col_idx)
    ws.merge_cells(f"{col_letter}3:{col_letter}5")
```

---

### 规则 2：分层列（扣款明细）— Row 3 水平合并

**适用条件**：列名含 `/`，第一段相同。

Row 3 只显示一级分类名，合并同一分组的所有列：

```
Q3:AA3 → 扣款明细（合并 17-27 列）
```

**代码**：按 `canonical_cols` 第一段分组，每组水平合并 row 3。

---

### 规则 3：分层列 — Row 4 水平合并

Row 4 显示二级分类名，合并第二段相同的列：

```
Q4:R4   → 养老    (扣款明细/养老/*)
S4:T4   → 失业    (扣款明细/失业/*)
U4:V4   → 医疗    (扣款明细/医疗/*)
W4      → 工伤险   (扣款明细/工伤险/*，仅一列)
X4:Y4   → 公积金   (扣款明细/公积金/*)
Z4      → 单位代理费 (扣款明细/单位代理费，仅一列)
AA4     → 扣款合计   (扣款明细/扣款合计，仅一列)
```

**代码**：按 `canonical_cols` 第二段分组，每组水平合并 row 4。

---

### 规则 4：分层列 — Row 5 **不合并**

Row 5 显示三级分类（单位/个人），**每个单元格独立**，不合并相同值。

```
Q5=单位  R5=个人  S5=单位  T5=个人
U5=单位  V5=个人  W5=单位  X5=单位  Y5=个人
```

即便 W5 和 X5 都是 "单位"，**也不合并**（源文件即是如此）。

---

### 规则 5：特殊列 — 垂直合并 {col}4:{col}5

部分扣款明细子列只有两级（没有三级），在 Row 4-5 做垂直合并：

```
Z4:Z5   → 单位代理费  (扣款明细/单位代理费)
AA4:AA5 → 扣款合计    (扣款明细/扣款合计)
```

这些列的 Row 3 = None（被 Q3:AA3 吸收），Row 4 显示值，Row 5 = None。

**判断条件**：列名只有 2 段（如 `扣款明细/单位代理费`），且 Row 5 无对应值。

---

## 判定流程总图

```
对于 canonical_cols 中的每一列 c:

  1. 列名不含 "/" → {c}3:{c}5 垂直合并（规则1）

  2. 列名含 "/":
     a. Row 3: 与同组第一段相同的列水平合并（规则2）
     b. Row 4: 与同组第二段相同的列水平合并（规则3）
     c. Row 5: 不合并任何单元格（规则4）
     d. 若列名只有 2 段（如 扣款明细/单位代理费）:
        → {c}4:{c}5 垂直合并（规则5）
```
